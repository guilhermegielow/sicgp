Passo 1)O arquivo WampServer2.0c.exe serve para instalar o servidor Apache, MySQL e o PHP.
Passo 2)O arquivo httpd.conf deve ser colocado na pasta C:\wamp\bin\apache\apache2.2.6\conf do servidor Apache.
Passo 3)O arquivo create sicgpi.sql deve ser rodado no phpmyadmin, cria todas as tabelas do sistema.
Passo 4)Após a criação das tabelas para acessar o sistema o usuário deve executar no phpmyadmin o script insert.sql afim de criar as permissões aos usuários do sistema.
